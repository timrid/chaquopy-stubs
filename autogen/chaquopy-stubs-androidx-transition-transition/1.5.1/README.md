# chaquopy-stubs-androidx-transition-transition

Python type stubs for [`androidx.transition:transition:1.5.1`](https://mvnrepository.com/artifact/androidx.transition/transition/1.5.1), generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.2.1`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.transition:transition:1.5.1` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash
pip install chaquopy-stubs-androidx-transition-transition==1.5.1
```
