# chaquopy-stubs-androidx-arch-core-core-runtime

Python type stubs for [`androidx.arch.core:core-runtime:2.0.1`](https://mvnrepository.com/artifact/androidx.arch.core/core-runtime/2.0.1), generated with [chaquopy-stubgen](https://github.com/timrid/chaquopy-stubgen) `0.2.2`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.arch.core:core-runtime:2.0.1` when using [Chaquopy](https://github.com/chaquo/chaquopy).

```bash
pip install chaquopy-stubs-androidx-arch-core-core-runtime~=2.0.0
```
