## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.versionedparcelable:versionedparcelable:1.1.1` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash

```
