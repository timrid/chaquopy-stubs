# chaquopy-stubs-androidx-core-core

Python type stubs for [`androidx.core:core:1.3.1`](https://mvnrepository.com/artifact/androidx.core/core/1.3.1), generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.2.1`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.core:core:1.3.1` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash
pip install chaquopy-stubs-androidx-core-core==1.3.1
```
