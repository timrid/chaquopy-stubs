# chaquopy-stubs-androidx-core-core

Python type stubs for [`androidx.core:core:1.15.0`](https://mvnrepository.com/artifact/androidx.core/core/1.15.0), generated with [chaquopy-stubgen](https://github.com/timrid/chaquopy-stubgen) `0.2.2`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.core:core:1.15.0` when using [Chaquopy](https://github.com/chaquo/chaquopy).

```bash
pip install chaquopy-stubs-androidx-core-core~=1.15.0
```
