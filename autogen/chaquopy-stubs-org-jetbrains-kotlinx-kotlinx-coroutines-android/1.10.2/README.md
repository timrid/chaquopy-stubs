Python type stubs for [`org.jetbrains.kotlinx:kotlinx-coroutines-android:1.10.2`](https://mvnrepository.com/artifact/org.jetbrains.kotlinx/kotlinx-coroutines-android/1.10.2), generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.2.1`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `org.jetbrains.kotlinx:kotlinx-coroutines-android:1.10.2` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash
pip install chaquopy-stubs-org-jetbrains-kotlinx-kotlinx-coroutines-android==1.10.2
```
