# chaquopy-stubs-android

Python type stubs for [`:android-35`](https://mvnrepository.com/artifact//android-35) version `35`, generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.1.1.dev49+g21c8ee3f3.d20260224`.

## Installation

```bash
pip install chaquopy-stubs-android==35
```

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `:android-35` when using [Chaquopy](https://chaquo.com/chaquopy/).
