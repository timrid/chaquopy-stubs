# chaquopy-stubs-android

Python type stubs for Android API Level 35, generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.2.1`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `Android API Level 35` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash
pip install chaquopy-stubs-android==35
```
