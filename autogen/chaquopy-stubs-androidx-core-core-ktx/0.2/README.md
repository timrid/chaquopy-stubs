# chaquopy-stubs-androidx-core-core-ktx

Python type stubs for [`androidx.core:core-ktx:0.2`](https://mvnrepository.com/artifact/androidx.core/core-ktx/0.2), generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.2.2`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.core:core-ktx:0.2` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash
pip install chaquopy-stubs-androidx-core-core-ktx~=0.2.0
```
