# chaquopy-stubs-androidx-annotation-annotation-experimental

Python type stubs for [`androidx.annotation:annotation-experimental:1.2.0`](https://mvnrepository.com/artifact/androidx.annotation/annotation-experimental/1.2.0), generated with [chaquopy-stubgen](https://github.com/timrid/chaquopy-stubgen) `0.2.2`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.annotation:annotation-experimental:1.2.0` when using [Chaquopy](https://github.com/chaquo/chaquopy).

```bash
pip install chaquopy-stubs-androidx-annotation-annotation-experimental~=1.2.0
```
