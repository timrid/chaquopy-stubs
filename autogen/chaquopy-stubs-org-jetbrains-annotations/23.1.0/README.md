# chaquopy-stubs-org-jetbrains-annotations

Python type stubs for [`org.jetbrains:annotations:23.1.0`](https://mvnrepository.com/artifact/org.jetbrains/annotations/23.1.0), generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.2.1`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `org.jetbrains:annotations:23.1.0` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash
pip install chaquopy-stubs-org-jetbrains-annotations==23.1.0
```
