# chaquopy-stubs-org-jetbrains-annotations

Python type stubs for [`org.jetbrains:annotations:24.0.1`](https://mvnrepository.com/artifact/org.jetbrains/annotations/24.0.1), generated with [chaquopy-stubgen](https://github.com/timrid/chaquopy-stubgen) `0.2.2`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `org.jetbrains:annotations:24.0.1` when using [Chaquopy](https://github.com/chaquo/chaquopy).

```bash
pip install chaquopy-stubs-org-jetbrains-annotations~=24.0.0
```
