# chaquopy-stubs-androidx-appcompat-appcompat

Python type stubs for [`androidx.appcompat:appcompat:1.4.2`](https://mvnrepository.com/artifact/androidx.appcompat/appcompat/1.4.2), generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.2.2`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.appcompat:appcompat:1.4.2` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash
pip install chaquopy-stubs-androidx-appcompat-appcompat==1.4.2
```
