# chaquopy-stubs-androidx-annotation-annotation

Python type stubs for [`androidx.annotation:annotation:1.0.1`](https://mvnrepository.com/artifact/androidx.annotation/annotation/1.0.1), generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.2.1`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.annotation:annotation:1.0.1` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash
pip install chaquopy-stubs-androidx-annotation-annotation==1.0.1
```
