# chaquopy-stubs-androidx-appcompat-appcompat-resources

Python type stubs for [`androidx.appcompat:appcompat-resources:1.7.1`](https://mvnrepository.com/artifact/androidx.appcompat/appcompat-resources/1.7.1), generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.2.1`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.appcompat:appcompat-resources:1.7.1` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash
pip install chaquopy-stubs-androidx-appcompat-appcompat-resources==1.7.1
```
