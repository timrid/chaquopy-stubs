# chaquopy-stubs-androidx-appcompat-appcompat-resources

Python type stubs for [`androidx.appcompat:appcompat-resources:1.6.1`](https://mvnrepository.com/artifact/androidx.appcompat/appcompat-resources/1.6.1), generated with [chaquopy-stubgen](https://github.com/timrid/chaquopy-stubgen) `0.2.2`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.appcompat:appcompat-resources:1.6.1` when using [Chaquopy](https://github.com/chaquo/chaquopy).

```bash
pip install chaquopy-stubs-androidx-appcompat-appcompat-resources~=1.6.0
```
