# chaquopy-stubs-androidx-fragment-fragment

Python type stubs for [`androidx.fragment:fragment:1.8.2`](https://mvnrepository.com/artifact/androidx.fragment/fragment/1.8.2), generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.2.2`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.fragment:fragment:1.8.2` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash
pip install chaquopy-stubs-androidx-fragment-fragment==1.8.2
```
