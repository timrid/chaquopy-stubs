# chaquopy-stubs-androidx-constraintlayout-constraintlayout-core

Python type stubs for [`androidx.constraintlayout:constraintlayout-core:1.0.4`](https://mvnrepository.com/artifact/androidx.constraintlayout/constraintlayout-core/1.0.4), generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.2.1`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.constraintlayout:constraintlayout-core:1.0.4` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash
pip install chaquopy-stubs-androidx-constraintlayout-constraintlayout-core==1.0.4
```
