# chaquopy-stubs-androidx-customview-customview

Python type stubs for [`androidx.customview:customview:1.0.0`](https://mvnrepository.com/artifact/androidx.customview/customview/1.0.0), generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.2.2`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.customview:customview:1.0.0` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash
pip install chaquopy-stubs-androidx-customview-customview==1.0.0
```
