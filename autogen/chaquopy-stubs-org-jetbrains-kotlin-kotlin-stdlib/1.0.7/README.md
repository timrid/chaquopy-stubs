# chaquopy-stubs-org-jetbrains-kotlin-kotlin-stdlib

Python type stubs for [`org.jetbrains.kotlin:kotlin-stdlib:1.0.7`](https://mvnrepository.com/artifact/org.jetbrains.kotlin/kotlin-stdlib/1.0.7), generated with [chaquopy-stubgen](https://github.com/timrid/chaquopy-stubgen) `0.2.2`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `org.jetbrains.kotlin:kotlin-stdlib:1.0.7` when using [Chaquopy](https://github.com/chaquo/chaquopy).

```bash
pip install chaquopy-stubs-org-jetbrains-kotlin-kotlin-stdlib~=1.0.0
```
