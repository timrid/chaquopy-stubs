# chaquopy-stubs-androidx-collection-collection

Python type stubs for [`androidx.collection:collection:1.4.0`](https://mvnrepository.com/artifact/androidx.collection/collection/1.4.0), generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.2.1`.

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.collection:collection:1.4.0` when using [Chaquopy](https://chaquo.com/chaquopy/).

```bash
pip install chaquopy-stubs-androidx-collection-collection==1.4.0
```
