# chaquopy-stubs-androidx-webkit-webkit

Python type stubs for [`androidx.webkit:webkit`](https://mvnrepository.com/artifact/androidx.webkit/webkit) version `1.15.0`, generated with [chaquopy-stubgen](https://github.com/chaquo/chaquopy) `0.1.1.dev49+g21c8ee3f3.d20260224`.

## Installation

```bash
pip install chaquopy-stubs-androidx-webkit-webkit==1.15.0
```

## Usage

These are stub-only packages ([PEP 561](https://peps.python.org/pep-0561/)). Install them alongside your project to get type checking support for `androidx.webkit:webkit` when using [Chaquopy](https://chaquo.com/chaquopy/).
